# sales-data-analysis
Sales data analysis using Python and data visualization
# Sales Data Analysis Dashboard

# Overview
This project analyzes sales data to identify trends and insights using Python.

# Technologies Used
- Python
- Pandas
- Matplotlib

# Features
- Monthly sales analysis
- Category-wise sales analysis
- Data visualization using bar and line charts

# Output
- Graphs showing sales trends
- Insights on top-performing categories

# Conclusion
This project helps in understanding business performance using data analysis.
